1. Структура проекта:

  index.html - главная страница сайта;
  event.html - шаблонная страница конкретного мероприятия;
  events.html - страница со списком мероприятий;
  seminar.html - шаблонная страница семинаров в альернативном дизайне;
  email/email.html - вёрстка email-письма;

  css/ - каталог с файлами стилей;
  fonts/ - каталог с используемыми шрифтами;
  img/ - каталог с изображениями;
  js/ - каталог со скриптами.

2. Описание проекта:

  Цель проекта: 
	Верстка адаптивного многостраничного сайта.
  
  Особенности и функционал:
	Сайт адаптирован под различные устройства, что обеспечивает удобство просмотра на мобильных устройствах и планшетах;
	Доболнительно к сайту реализован шаблон для email-писем.

  Используемые технологии: 
	HTML5, CSS3 (Sass), JavaScript, GULP.
 
  Результаты и достижения: 
	В результате сверстаны все страницы многостраничного сайта для дальнейшей интегрирации компанию.
