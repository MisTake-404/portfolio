1. Структура проекта:

  index.html - главная страница сайта;
  fag.html - страница "Вопрос-ответ";
  manual.html - страница  "Инструкция";
  reviews.html - страница  "Отзывы";
  css/ - каталог с файлами стилей;
  fonts/ - каталог с используемыми шрифтами;
  img/ - каталог с изображениями;
  js/ - каталог со скриптами.

2. Описание проекта:

  Цель проекта: 
	Верстка адаптивного лендинга для компании, специализирующейся на создании контрацептивного средства.
  
  Особенности и функционал: 
	Верстка главной страницы подстроена под плагин FullPage (создания полноэкранных прокручиваемых секций на веб-странице);
	Внедрены современные анимации при помощи плагина Lottie;
	Сайт адаптирован под различные устройства, что обеспечивает удобство просмотра на мобильных устройствах и планшетах.

  Используемые технологии: 
	HTML5, CSS3 (Sass), JavaScript, jQuery, GULP, Плагины js FullPage и Lottie.
 
  Результаты и достижения: 
	В результате разработаны основные страницы сайта для дальнейшей интегрирации в бизнес-процессы компании.
