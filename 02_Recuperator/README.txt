1. Структура проекта:

  index.html - главная страница сайта;
  policy.html - страница "Политика конфиденциальности";
  css/ - каталог с файлами стилей;
  fonts/ - каталог с используемыми шрифтами;
  img/ - каталог с изображениями;
  js/ - каталог со скриптами.

2. Описание проекта:

  Цель проекта: 
	Верстка адаптивного лендинга для компании, специализирующейся на производстве косметического средства.
  
  Особенности и функционал:
	Верстка главной страницы содержит несколько разнообразных слайдеров;
	Сайт адаптирован под различные устройства, что обеспечивает удобство просмотра на мобильных устройствах и планшетах.

  Используемые технологии: 
	HTML5, CSS3 (Sass), JavaScript, GULP, Плагины js Swiper и Fancybox (min).
 
  Результаты и достижения: 
	В результате разработан лендинг главной страницы сайта для дальнейшей интегрирации в бизнес-процессы компании.
